# Design System

## Direcao visual

- premium e leve
- moderno sem parecer frio
- menos blocos na primeira dobra
- mais foco na leitura principal
- contraste suficiente sem visual pesado

## Tokens principais

- fundo: `#eff3f8`
- superficie: `rgba(255,255,255,0.9)`
- texto principal: `#15253a`
- texto secundario: `#67768b`
- acento: `#148475`
- destaque quente: `#c99042`
- sucesso: `#1b8d61`
- alerta: `#c7802b`
- erro: `#cf5449`

## Tipografia

- sans: `Aptos`, `Segoe UI Variable Text`, `Segoe UI`
- serif editorial para titulos: `Iowan Old Style`, `Palatino Linotype`

## Componentes base

- cards com cantos amplos e sombra suave
- sidebar escura com navegacao clara
- topbar leve com fundo transluscido
- formularios com borda fina e foco verde agua
- tabelas respiráveis com cabecalho discreto
- toasts compactos
- estados vazios com copy curta

## Padroes de experiencia

- linguagem direta em portugues do Brasil
- acao principal sempre evidente
- filtros claros e sem excesso
- feedback imediato em mutacoes
- status visuais simples: compensada, planejada, atrasada
